RMImageScroller by robotmedia

Image scroller for iOS

RMImageScroller is an image scroller for iOS 3.2 or later. It is typically used to show thumbnails for a sequential viewer. It supports iPhone, iPad and all screen orientations.

Dependencies

iOS 3.2 or later

Install

  1. Copy the contents of /Classes to your project. You will need both RMImageScroller and RMUIUtils (.h and .m).
  2. Import RMImageScroller.h in the necessary files.
  3. Provide a delegate for the scroller that conforms to the RMImageScrollerDelegate protocol defined in RMImageScroller.h. Refer to RMImageScrollerViewController of the sample project for an implementation example.
